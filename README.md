# Codveda Technology - ML Internship

## About
Machine Learning Internship tasks completed as part of Codveda Technology's internship program.

## Level 1 - Basic

### Task 1: Data Preprocessing
- Dataset: Iris Dataset
- Handled missing values (mean imputation)
- Label encoding for categorical variable (species)
- Standardization using StandardScaler
- Train/Test split (80/20)

### Task 2: Linear Regression
- Dataset: Boston House Prediction Dataset
- Built Linear Regression model using scikit-learn
- Evaluated using R² (0.67) and MSE (24.29)

## Tools Used
- Python
- Pandas
- Scikit-learn
- NumPy
